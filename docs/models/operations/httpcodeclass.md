# HttpCodeClass

Filter by HTTP response code class (2xx, 3xx, 4xx, 5xx).

## Example Usage

```ruby
require "spaire"

value = HttpCodeClass::TWOXX
```


## Values

| Name      | Value     |
| --------- | --------- |
| `TWOXX`   | 2xx       |
| `THREEXX` | 3xx       |
| `FOURXX`  | 4xx       |
| `FIVEXX`  | 5xx       |
